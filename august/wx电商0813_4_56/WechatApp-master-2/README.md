###概述

简单的电商系统小程序开发实践

- /lib/request.js       请求封装方法js
- /lib/resource.js      请求资源封装js
- /lib/promise__        引入promise
- /lib/city             引入城市选择控件
- /templates            小程序模版目录
- /wxParse              小程序html解析扩展

###使用

克隆本项目 -> 在微信开发工具中添加项目 -> 选择项目app目录
